Default site password:

To change the password:
Open script.js and edit this line:
const SITE_PASSWORD = 'BGW2026';

Important privacy note:
This is a simple front-end password screen. It is fine for casual privacy among friends, but it is not high-security. For stronger privacy, use Netlify password protection, Cloudflare Access, or a member-login tool.

To publish on Netlify:
1. Go to netlify.com
2. Create a free account
3. Drag the bgw-website folder into Netlify's deploy area
4. Netlify will give you a live URL
5. Update the URL/name later if desired
